# 命令模板

python3 ocr_cli.py \
  --input-dir "/home/featurize/data" \
  --pattern "*.png" \
  --out "/home/featurize/work/中国通俗小说鉴赏词典.md" \
  --model PP-OCRv5_mobile \
  --batch-size 10 \
  --device gpu \
  --page-range 16-1371


解释：
--out 参数必须设定为一个markdown文档，不能是路径。
RTX 3060 12G 显存不能运行 PP-OCRv5_server。
最好指定页码，这样更精准。

注意：
一次处理1371页，耗时两小时左右。
内存占用逐渐增加至：73%
CPU和GPU占用平稳
